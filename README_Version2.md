# Phaser RPG Prototype

A beginner-friendly 2D top-down RPG made with Phaser 3 and pure JavaScript!

## Getting Started

### Option 1: Play Online (Easiest)
1. Go to your GitHub repository settings
2. Enable **GitHub Pages** (set source to main branch)
3. Your game will be live at: `https://PurpleOctopus49.github.io/phaser-rpg-prototype/`

### Option 2: Play Locally
1. Clone this repository
2. Open `index.html` in your web browser
3. That's it! No installation needed

## Controls

- **Arrow Keys** or **WASD**: Move your character
- **SPACE**: Interact with NPCs

## Game Features

✅ Player character (green square)
✅ Arrow key and WASD movement
✅ Obstacle collisions (trees)
✅ NPC characters (magenta and yellow)
✅ Dialog system - talk to NPCs!
✅ Camera follows player
